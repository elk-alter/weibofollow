## 功能列表

- 微博用户数量极其庞大
- 用户可以关注自己的喜欢的用户
- 用户取消关注已关注的用户
- 用户可以看见自己关注了谁
- 用户可以看见谁关注了自己
- 查询用户是否关注了另一个用户
